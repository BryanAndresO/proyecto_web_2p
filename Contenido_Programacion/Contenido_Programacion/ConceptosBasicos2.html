Manejo de Cadenas (String)

Para entenderlas mejor, imagina que una cadena es como una fila de letras, donde cada letra tiene una posición.

    Ejemplo:

    Cadena: "Hola"

    Posiciones: 0, 1, 2, 3

    H → Posición 0 → Contador 1
    o → Posición 1 → Contador 2
    l → Posición 2 → Contador 3
    a → Posición 3→ Contador 4
