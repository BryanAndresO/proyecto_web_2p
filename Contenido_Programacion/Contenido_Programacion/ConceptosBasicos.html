Conceptos Básicos

Los algoritmos realizan operaciones simples como lectura y escritura de datos, operaciones aritméticas y comparaciones. También incluyen estructuras de control como condicionales y ciclos para tomar decisiones y repetir acciones.

Operaciones Básicas

Los algoritmos suelen realizar operaciones simples como:

1. Lectura de datos

Obtener datos de una fuente, como el usuario o un archivo. Por ejemplo:

dato = leer("Escribe un número: ")

2. Escritura de datos

Mostrar resultados al usuario o guardar en un archivo. Por ejemplo:

escribir("El resultado es:", resultado)

3. Operaciones aritméticas

Sumar, restar, multiplicar, dividir, calcular el módulo (resto de una división). Por ejemplo:

suma = 5 + 3
resta = 7 - 2
multiplicacion = 4 * 6
division = 8 / 2
modulo = 9 % 4

4. Comparaciones

Operadores de comparación para evaluar relaciones entre valores:

igual = (5 == 5) # Igualdad
diferente = (5 != 3) # Desigualdad
mayor = (7 > 3) # Mayor que
menor = (3 < 7) # Menor que
mayor_o_igual = (7 >= 5) # Mayor o igual que
menor_o_igual = (5 <= 7) # Menor o igual que

    Variables y Tipos de Datos

    Las variables son contenedores que almacenan información, como números o texto. Los tipos de datos definen el tipo de valor que una variable puede almacenar.

    Tipos de datos:

    1. Enteros (int)

    Los enteros representan números enteros sin decimales. Pueden ser positivos o negativos. Por ejemplo:

    numero_entero = 10

    2. Números de punto flotante (float)

    Los números de punto flotante representan números con decimales. Por ejemplo:

    numero_flotante = 3.14

    3. Cadenas de texto (str)

    Las cadenas de texto representan secuencias de caracteres encerradas entre comillas simples o dobles. Por ejemplo:

    mensaje = "¡Hola, mundo!"

    4. Booleanos (bool)

    Los booleanos representan valores de verdadero (True) o falso (False). Son útiles en lógica y control de flujo. Por ejemplo:

    verdadero = True
    falso = False

    Condicionales y Ciclos

    Los condicionales permiten tomar decisiones basadas en condiciones, mientras que los ciclos repiten acciones varias veces. Ambos son fundamentales para el control del flujo en los programas.

    Condicionales
