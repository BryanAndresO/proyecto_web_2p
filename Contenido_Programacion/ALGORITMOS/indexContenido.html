¿Qué es un Algoritmo?

Un algoritmo es una serie de pasos organizados que se utilizan para resolver un problema o realizar una tarea.

Es como una receta de cocina: sigues las instrucciones paso a paso para obtener un resultado.

Conceptos Fundamentales del Algoritmo

Veamos los conceptos teóricos básicos del algoritmo usando una receta de cocina para mejor entendimiento.

  • Entrada: Puedes pensar en esto como los ingredientes en una receta de cocina.
  • Proceso: El proceso es donde ocurre la magia. Siguiendo con nuestra analogía de la receta, esto sería como mezclar los ingredientes y hornear la mezcla para hacer una galleta.
  • Salida: Es lo que obtienes después de seguir todos los pasos del proceso. En nuestra receta, la salida serían las deliciosas galletas terminadas.
  • Secuencia: Si mezclas los ingredientes en el orden equivocado, es posible que no obtengas lo que esperabas. La secuencia es vital para que el algoritmo funcione correctamente.
  • Iteración (Bucles): significa repetir algunos pasos varias veces. Imagina que estás haciendo muchas galletas; podrías repetir los pasos de “poner la masa en la bandeja” y “hornear” hasta que hayas hecho todas las galletas.
  • Eficiencia: Algunas recetas son más rápidas y fáciles de hacer que otras. De la misma manera, algunos algoritmos resuelven problemas más rápidamente que otros.

Conceptos Básicos

Así como sigues pasos en una receta o los pasos de la vida diaria, un algoritmo son los pasos que una computadora sigue para resolver un problema.

Un algoritmo necesita instrucciones claras y fáciles de seguir, como cuando te dicen: “Primero haces esto, luego haces aquello”.

Operaciones Básicas

Los algoritmos suelen realizar operaciones simples como leer datos, procesarlos y mostrar resultados, así como también realizar cálculos simples como sumar, restar, multiplicar o dividir

  • Lectura de datos: Obtener datos de una fuente, como el usuario o un archivo.
  • Escritura de datos: Mostrar resultados al usuario o guardar en un archivo.
  • Operaciones aritméticas: Sumar, restar, multiplicar, dividir, calcular el módulo (resto de una división).
  • Comparaciones: Igualdad (==), desigualdad (!=), mayor que (>), menor que (<), mayor o igual que (>=), menor o igual que (<=).

Condicionales

Permiten tomar decisiones dependiendo de ciertas condiciones.

  • Condicional simple: Ejecuta una acción si se cumple una condición.

    • Si condición Entonces

    FinSi

  • Condicional compuesta: Incluye una acción alternativa si no se cumple la condición inicial.

    • Si condición Entonces

    sino

    FinSi

Ciclos

Sirven para repetir acciones un cierto número de veces.

  • Ciclo Mientras: Repite las acciones mientras se cumpla una condición.

    • Mientras condición Hacer

    acciones

    FinMientras

  • Ciclo Para: Repite las acciones un número específico de veces.

    • Para i <- inicio Hasta fin Hacer

    acciones

    sino

    FinPara

Factorial de un número

El factorial (“n!”) es el producto de todos los números desde 1 hasta “n”.

  • Calcular el factorial de “n”

    • Leer "n".

    factorial <- 1.

    Para i <- 1 Hasta n Hacer factorial <- factorial * i

    FinPara

    Escribir "El factorial es:", factorial.

Serie de Fibonacci

La Serie de Fibonacci comienza con 0 y 1; cada término siguiente es la suma de los dos anteriores.

  • Generar la serie hasta el término “n”

    • Leer un número (“n”).

    Inicializar “a = 0”, “b = 1”.

    Imprimir “a”, “b”.

    Para cada término desde 3 hasta “n”:

    c = a + b.

    Imprimir c.

    Actualizar a = b, b = c.

    FinPara

    • Leer una cadena cadena.

    Inicializar contador = 0.

    Para cada carácter en cadena:

    Incrementar contador en 1.

    Imprimir contador.

  • Verificar si una palabra es un palíndromouna (palabra que se lee igual de izquierda a derecha y de derecha a izquierda).

    Comparar palabra con su inversa.

    Si son iguales, imprimir "Es un palíndromo".

Arreglos (Vectores y Matrices)

  • Operaciones con vectores (suma de los elementos de un vector )

    • Leer el tamaño del vector n.

    Leer los n elementos del vector A.

    Inicializar suma = 0.

    Para i = 1 hasta n:

    suma = suma + A[i].

    Imprimir suma.

  • Operaciones con matrices (suma de los elementos de dos matrices)

    • Leer el tamaño de las matrices n × m.

    Leer los elementos de la matriz A.

    Leer los elementos de la matriz B.

    Inicializar una matriz C de n × m.

    Para i = 1 hasta n:

    Para j = 1 hasta m:

    C[i][j] = A[i][j] + B[i][j].

    Imprimir la matriz C.

  • Búsquedas y ordenamientos en arreglos

    • Búsqueda lineal en un vector: Este algoritmo realiza una búsqueda lineal para encontrar un número en un vector.

    Leer el tamaño del vector n.

    Leer los n elementos del vector A.

    Leer el número x a buscar.

    Para i = 1 hasta n:

    Si A[i] = x, imprimir "Encontrado en posición i" y terminar.

    Si no se encuentra, imprimir "No encontrado".
