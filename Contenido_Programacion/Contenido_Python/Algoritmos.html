Algoritmos en Python

Explora una colección de algoritmos básicos para aprender y practicar Python, desde operaciones matemáticas simples hasta ciclos y estructuras condicionales.

Suma

1. Leer a.
2. Leer b.
3. Calcular suma = a + b.
4. Imprimir suma.

Resta

1. Leer a.
2. Leer b.
3. Calcular resta = a - b.
4. Imprimir resta.

Multiplicación

1. Leer a.
2. Leer b.
3. Calcular producto = a * b.
4. Imprimir producto.

División

1. Leer a.
2. Leer b.
3. Calcular cociente = a / b.
4. Imprimir cociente.

Condicional Simple

1. Leer n.
2. Si n > 0 entonces
3. Imprimir "El número es positivo".

Fibonacci

1. Leer n.
2. Inicializar a = 0, b = 1.
3. Imprimir a y b.
4. Para i desde 3 hasta n, hacer:
- Calcular c = a + b.
- Imprimir c.
- Actualizar a = b y b = c.
